static void aery::gpio_enable_localbus ( void  )
inlinestatic

Enables GPIO local bus

Before GPIO ports can be accessed through local bus you have to call this function. Note that when local bus has been enabled you cannot access GPIO ports through PBB (Peripheral Bus B) anymore, which was the default bus accessing them, and thus functions such as gpio_set_pin_high() or gpio_read_pin() has no effect. From now on you have to use bitbanging and instead of using, for example, porta->ovr you have to use new expression, lporta->ovr. To reroute GPIO back to PBB call function gpio_disable_localbus().

Attention:
CPU clock has to match with PBB clock to make local bus functional

Initializes the voltage controlled oscillator (VCO) the given PLL

Parameters:
ppllPointer to the PLL register
srcSource oscillator
mulThe source frequency is multiplier by this value
divThe source frequency is divided by this value
hifreqIf true enables the high frequency mode
Equations:
  • If div > 0, then f_vco = f_src * mul / div
  • If div = 0, then f_vco = 2 * f_src * mul

void aery::spi_setup_npcs ( volatile avr32_spi_t *  pspi,
uint8_t  npcs,
enum Spi_mode  mode,
uint8_t  bits 
)

Setups the SPI mode and the shift register width of the Numeric Processor Chip Select (NPCS, same as slave/chip select)

Parameters:
pspiPointer to the SPI peripheral
npcsChip Select: 0-3
modeSPI mode
bitsWidth of SPI shift register: 8-16 bits
Making SPI SCK faster
Chip select baudrate is hard coded to MCK/255. To make it faster you can bitbang the SCRB bit in CSRX register.
spi_setup_npcs(&AVR32_SPI0, 0, SPI_MODE1, 16);
spi0->CSR0.scbr = 32; // baudrate is now MCK/32
Note:
Every chip select line can have different mode and size of shift register.
Examples:
displays/hd44780.cpp, and spi.cpp.
