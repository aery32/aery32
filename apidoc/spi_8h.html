void aery_spi_init_master ( volatile avr32_spi_t *  )

Init SPI as a master

Init CS multiplexer
If you are using multiplexed chip selects, enable cs multiplexing by bitbanging PCSDEC bit in SPI MR register after initialization: 
 aery_spi_init_master(&AVR32_SPI0);
 AVR32_SPI0.MR.pcsdec = 1;

Setup SPI mode and the shift register width of the Numeric Processor Chip Select (NPCS, same as slave select)

Making SPI SCK faster
Chip select baudrate is hard coded to MCK/128. To make it faster you can bitbang the SCRB bit in CSR register. But be aware of the errata of SPI bad serial clock generation, see datasheet page 794. 
 aery_spi_setup_chipselect(AVR32_SPI0.csr0, SPI_MODE1, 16);
 AVR32_SPI0.csr0.scbr = 32; // baudrate is MCK/32
Parameters:
pspiPointer to the SPI peripheral
npcsChip Select
modeSPI mode
bitsWidth of SPI shift register: 8-16 bits
