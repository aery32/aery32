enum Pm_pll_source  src,
uint8_t  mul,

bool  hifreq 
)

Initializes the voltage controlled oscillator (VCO) the given PLL

Parameters:
ppllPointer to the PLL register
srcSource oscillator
mulThe source frequency is multiplier by this value
divThe source frequency is divided by this value
hifreqIf true enables the high frequency mode
Equations:
  • If div > 0, then f_vco = f_src * mul / div
  • If div = 0, then f_vco = 2 * f_src * mul
Note:
VCO frequency has to be in range of 80 MHz - 180 MHz or if hifreq mode is enabled in the range of 160 MHz - 240 MHz.
