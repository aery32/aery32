Detailed Description

String functions.

Every now and then sprintf/sscanf family of functions take too much space of memory or do not even work because of problems like stack overflow. Thus Aery32 Framework provides a bunch of small string functions.

EXPERIMENTAL!!! These functions may be removed or heavily changed.
